# VIM 之窗口操作

```bash
[ CTRL-W s ]        上下 切分窗口
[ CTRL-W v ]        左右 切分窗口

[ CTRL-W = ]        让所有窗口 宽高相同
[ CTRL-W _ ]        纵向 最大化 当前窗口
[ CTRL-W | ]         横向 最大化 当前窗口

[ CTRL-W p ]        跳到上一个访问过的窗口
[ CTRL-W w ]       循环切换到下一个窗口
[ CTRL-W W ]       循环切换到上一个窗口
[ CTRL-W x ]        交换窗口
[ CTRL-W R ]        旋转窗口
[ CTRL-W T ]        将当前窗口移到新的标签页中

[ CTRL-W c ]        关闭当前窗口
[ CTRL-W o ]        关闭其他窗口

[ CTRL-W h ]        跳到 左边的窗口
[ CTRL-W j ]         跳到 下边的窗口
[ CTRL-W k ]        跳到 上边的窗口
[ CTRL-W l ]         跳到 右边的窗口
[ CTRL-W H ]        将当前窗口 移动到 最左边
[ CTRL-W J ]        将当前窗口 移动到 最下边
[ CTRL-W K ]        将当前窗口 移动到 最上边
[ CTRL-W L ]        将当前窗口 移动到 最右边

[ CTRL-W +或- ]     增加/减少当前窗口的行高，前面可以加数字
[ CTRL-W <或> ]     减少/增加当前窗口的列宽，前面可以加数字
```
