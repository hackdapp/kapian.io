# VIM 之插入模式

![](http://cdn.hackdapp.com/2021-01-28-20200722%20-%20VIM%E4%B9%8B%E6%8F%92%E5%85%A5%E6%A8%A1%E5%BC%8F.jpg)

[ i ] 在光标处, 进入插入模式；
[ I ] 在行首, 进入插入模式；
[ a ] 在光标后, 进入插件模式；
[ A ] 在行尾, 进入插入模式；
[ o ] 在下一行插入新行，进入插入模式；
[ O ] 在 上一行插入新行，进入插入模式；
[ gi ] 进入上一次插入模式的位置；
< ESC > 退出插入模式

说明：[command]中括号中的代码 command 为 VIM 命令，可直接在 VIM 编辑器中运行。
